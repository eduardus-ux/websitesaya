Contoh Soal SPLDV

Diberikan sistem persamaan berikut:

  • 2x + y = 8
  • x - y = 2

Selesaikan sistem persamaan di atas menggunakan tiga metode berikut:

1. Metode Eliminasi

Langkah-langkah:

  1. Eliminasi variabel y: jumlahkan kedua persamaan:

    2. (2x + y) + (x - y) = 8 + 2

    3x = 10, maka x = 10/3.

  2. Substitusi x = 10/3 ke salah satu persamaan, misalnya x - y = 2:

    3. 10/3 - y = 2, maka y = 10/3 - 6/3 = 4/3.

Hasil: x = 10/3, y = 4/3.

2. Metode Substitusi

Langkah-langkah:

  1. Ekspresikan y dari persamaan kedua: x - y = 2.

    2. y = x - 2.

  2. Substitusi y = x - 2 ke persamaan pertama:

    3. 2x + (x - 2) = 8.

    3x - 2 = 8, maka 3x = 10 dan x = 10/3.

  3. Substitusi x = 10/3 ke y = x - 2:

    4. y = 10/3 - 6/3 = 4/3.

Hasil: x = 10/3, y = 4/3.

3. Metode Grafik

Langkah-langkah:

  1. Gambar grafik kedua persamaan:

    2. Persamaan 1: 2x + y = 8. Nilai-nilai:

    • Jika x = 0, maka y = 8.
    • Jika y = 0, maka x = 4.

    Persamaan 2: x - y = 2. Nilai-nilai:

    • Jika x = 0, maka y = -2.
    • Jika y = 0, maka x = 2.
  2. Plot kedua garis di grafik dan cari titik potongnya.

Titik potong: (10/3, 4/3).
